# üéÑ Advent of Code | {year=>2025}

This repository contains my solutions for the [Advent of Code 2025](https://adventofcode.com/2025) challenges, implemented in **Go (Golang)**.

## üöÄ Usage

```bash
    # Add your input.txt file in the folder of the day you want to run

    # run specific day (x = number of day)
    $ make run-day day=x
    
    # run all days
    $ make run-all
    
    # Also you run directly using go:
    $ go run <day>/main.go
```
